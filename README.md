# DATTU Quantum CRM

## Project Overview

DATTU Quantum CRM is an AI-powered B2B multi-product CRM application built with React, TypeScript, Node.js, and Prisma.

## Tech Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn-ui
- Node.js
- Express
- Prisma

## Local Development

1. Clone the repository.
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the app (frontend + backend):
   ```sh
   npm run dev
   ```

## Build

```sh
npm run build
```

## Environment Setup

Copy `.env.example` values into your local `.env` and set real credentials for your environment.

## Deployment

Deploy the frontend and backend using your existing platform setup (for example Vercel and Render).
